# EY (EULARIYANI)

**EY** is a mobile application built with React Native and Expo, designed to help users take full control of their lives through discipline, time management, and self-awareness.

---

## 🧠 Concept

EY is not just a productivity app.

It is a **life control system** that helps users:

* ⏳ Manage their time effectively
* ✅ Stay consistent with daily tasks
* 📊 Track discipline and performance
* 🎯 Build better habits
* ❤️ (Future) Improve relationship accountability
* 💰 (Future) Manage personal finances

---

## 🎯 Current Features (MVP)

* 📱 Built with React Native + Expo
* 🧾 Daily task system (max 3 tasks per day)
* 📅 Basic planning structure
* 📊 Weekly discipline tracking (in progress)
* 🔄 Scalable architecture for future features

---

## 🏗 Tech Stack

* **React Native (Expo)**
* **TypeScript**
* **React Navigation**
* **Zustand (State Management)**
* **Firebase (Planned for backend)**
